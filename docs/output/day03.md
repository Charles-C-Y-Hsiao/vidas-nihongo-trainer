# Day 03 - 搜尋與篩選案件

⸻

🔹 ① 原句

👉 キーワードを入力して、関連する案件を検索します。

⸻

🔹 ② 快速閱讀版（Chunk / 拆解）

目的：

👉 一邊閱讀

👉 一邊理解

👉 不需要上下滑動

👉 Chunk 與拆解放在一起

──────────────────

Chunk ①

* キーワードを入力して
* きーわーどをにゅうりょくして
* 輸入關鍵字後
* kiiwaado o nyuuryoku shite

拆解 ①

* キーワード（きーわーど）= 關鍵字（kiiwaado）
* を = 動作的對象（o）
* 入力する（にゅうりょくする）= 輸入（nyuuryoku suru）
* して = 做完後接下一個動作（shite）

──────────────────

Chunk ②

* 関連する案件を検索します
* かんれんするあんけんをけんさくします
* 搜尋相關案件
* kanren suru anken o kensaku shimasu

拆解 ②

* 関連する（かんれんする）= 相關的（kanren suru）
* 案件（あんけん）= 案件（anken）
* を = 動作的對象（o）
* 検索します（けんさくします）= 搜尋（kensaku shimasu）

──────────────────

👉 整句快速理解：

輸入關鍵字後，搜尋相關案件。

⸻

🔹 ③ 句尾判斷（⚡最優先）

最後真正結尾：

* 検索します

👉 中文意思：搜尋

🔹 ① 原句

👉 条件に一致する案件がない場合は、メッセージを確認してください。

⸻

🔹 ② 快速閱讀版（Chunk / 拆解）

目的：

👉 一邊閱讀

👉 一邊理解

👉 不需要上下滑動

👉 Chunk 與拆解放在一起

──────────────────

Chunk ①

* 条件に一致する案件がない場合は
* じょうけんにいっちするあんけんがないばあいは
* 如果沒有符合條件的案件
* jouken ni icchi suru anken ga nai baai wa

拆解 ①

* 条件（じょうけん）= 條件（jouken）
* に = 符合的基準（ni）
* 一致する（いっちする）= 符合、一致（icchi suru）
* 案件（あんけん）= 案件（anken）
* が = 主語（ga）
* ない = 沒有（nai）
* 場合（ばあい）= 情況（baai）
* は = 提示主題（wa）

──────────────────

Chunk ②

* メッセージを確認してください
* めっせーじをかくにんしてください
* 請確認訊息
* messeeji o kakunin shite kudasai

拆解 ②

* メッセージ（めっせーじ）= 訊息（messeeji）
* を = 動作的對象（o）
* 確認してください（かくにんしてください）= 請確認（kakunin shite kudasai）

──────────────────

👉 整句快速理解：

如果沒有符合條件的案件，請確認系統訊息。
