# Day 04 - 更新案件狀態

⸻

🔹 ① 原句

👉 案件詳細ページを開いて、現在の状態を確認します。

⸻

🔹 ② 快速閱讀版（Chunk / 拆解）

目的：

👉 一邊閱讀

👉 一邊理解

👉 不需要上下滑動

👉 Chunk 與拆解放在一起

──────────────────

Chunk ①

* 案件詳細ページを開いて
* あんけんしょうさいページをひらいて
* 開啟案件詳細頁
* anken shousai peeji o hiraite

拆解 ①

* 案件詳細ページ（あんけんしょうさいページ）= 案件詳細頁（anken shousai peeji）
* を = 動作對象（o）
* 開いて（ひらいて）= 開啟後、打開後（hiraite）

──────────────────

Chunk ②

* 現在の状態を確認します
* げんざいのじょうたいをかくにんします
* 確認目前狀態
* genzai no joutai o kakunin shimasu

拆解 ②

* 現在（げんざい）= 現在、目前（genzai）
* の = 連接名詞，表示「的」（no）
* 状態（じょうたい）= 狀態（joutai）
* を = 動作對象（o）
* 確認します（かくにんします）= 確認（kakunin shimasu）

──────────────────

👉 整句快速理解：

開啟案件詳細頁，確認目前的案件狀態。

⸻
